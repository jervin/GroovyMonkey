Installing Groovy Monkey

Groovy Monkey is installed using the Eclipse Update Manager pointing to the Groovy Monkey Sourceforge update site http://groovy-monkey.sourceforge.net/update.

Simple Instructions

1. Open the Update Manager.

2. Create a Groovy Monkey remote update site. (Select it and press "Finish".)

3. Select the Groovy Monkey feature. (Use "Next", agree to various licenses, and press "Finish".)
