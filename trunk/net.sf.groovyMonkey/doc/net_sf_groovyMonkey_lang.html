  • name - an optional name of the extension instance

<!ELEMENT lang EMPTY>

<!ATTLIST lang

id      CDATA #IMPLIED

name    CDATA #REQUIRED

factory CDATA #REQUIRED

>

  • id - specifies a unique identifier for this extension
  • name - The short name of the scripting language that this extension is providing to Eclipse Monkey. Must match the language metatag given in the monkey script.
  • factory - The factory that generates the Script Runners for the given scripting language.
