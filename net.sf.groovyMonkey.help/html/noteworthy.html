New & Noteworthy

Version 0.2.0 - August 2, 2006

  • Now using the Bean Scripting Framework (BSF) to simplify adding JVM scripting languages to Groovy Monkey.
  • Added Job and Exec-Mode metatags that show that all monkey scripts run from within an Eclipse Job.
  • Added Groovy Monkey Editor that contains a content outline, run script action, and add dom action.
  • Added outline view that shows the metadata that the Groovy Monkey runtime will see and the types available in the binding.
  • Enabled direct navigation via double click from the outline view.
  • Added Installed DOMs view to list the DOMs that are currently installed in the workbench.
  • Added the Listener metadata tag to enable scripts to be able to respond to ResourceChangeEvent(s) in the current workspace.
  • Added the Runner DOM to allow monkey scripts to invoke other monkey scripts.

Version 0.2.1

  • Changed the Groovy Monkey default project name to GroovyMonkeyScripts as per issue: [ 1536760 ] Change GroovyMonkeyExamples to GroovyMonkeyScripts
  • Added an Include-Bundle dialog to the editor context menu, to the Package Explorer view and Navigator.
  • Added Include and Include-Bundle dialogs to Groovy Editor context menu command set.
